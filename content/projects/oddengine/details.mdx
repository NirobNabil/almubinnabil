# OddEngine - a 2D game engine built with OpenGL and custom physics engine in Java

> We Built a 2D game engine from scratch using java and OpenGL with GPU optimized 2D physics engine

<Br/>

Github repository:
[https://github.com/nirobNabil/oddengine](https://github.com/nirobNabil/oddengine)

<Br/>


The Graphics pipeline doesnt support loading meshes or sprites, rather user has to create shapes using triangles. 
The physics engine supports the following features:
- Rigid body
- Bouncing
- Collision detection and response
- Gravity
- Friction
- Joints (mouse joint)
