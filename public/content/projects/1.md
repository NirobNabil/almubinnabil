# Description
Complete redesign of our core dashboard interface. Focused on improving information hierarchy and reducing cognitive load through brutalist design principles.

Key achievements:
- 40% reduction in user errors
- Improved task completion time
- Unified design system across product

Process involved extensive user research, iterative prototyping, and cross-functional collaboration with engineering and product teams.
